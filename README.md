# 3-State-Busy-Beaver
Simple Java program that calculates 3 state busy beaver (you can find state table I used here: https://en.wikipedia.org/wiki/Turing_machine)
# How to compile & run
## Windows
1. Open cmd
2. Go to folder where you put .java file
3. Type in:
```
javac Main.exe
```
4. To run type:
```
java Main
```
